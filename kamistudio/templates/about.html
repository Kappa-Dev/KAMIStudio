About the project

{% endblock %} {% block inner_workspace %}

KAMIStudio is a bio-curation environment for modelling cellular signalling developed at the ENS de Lyon. KAMIStudio is a part of the bio-curation project KAMI and is based on the KAMI (Knowledge Aggregator and Model Instantiator) library. It is also a part of a larger Kappa project.
KAMI uses graph-based knowledge representation system and allows to curate corpora of de-contextualized cellular signalling knowledge. It provides capabilities for semi-automatic aggregation of individual protein-protein interactions and their instantiation in diffenent contexts. It also allows the user to automatically generate Kappa rules from instantiated signalling model.
The meta-model of KAMI describes the kinds of entities that can be represented in KAMIStudio.

KAMI meta-model


You can find more about the knowledge representation system used in KAMI in the following article.

KAMIStudio is developed as a web-based application and can be run locally. Source code and instuctions for installing locally can be found in the GitHub repository. Please, report any issues by the link.
